Описание демона контролирующего индикатор на передней панели.
Запуск программы:
	frontpanel [OPTIONS]
		-d - не демонизировать процесс,
		-q - тихий режим. Отключить отображение времени по умолчанию. Ничиге не виводить при завершении программы.

Программа стартует из автозагрузочных скриптов (/etc/init.d/S95leds в rootfs и /etc/init.d/S13frontpanel.sh в initramfs).
Демон при старте должен проверить переменную board_env, которая содержит тип привтавки (тоже самое можно сделать разобрав файл /proc/cmdline).
Демон должен запускаться на приставках STB840 AVIT или Промсвязь (stb840_ch7162, stb840_promSvyaz).
Для приставки STB840 AVIT надо проверить наличие файл /sys/devices/platform/ct1628/text, а для Промсвязи файл /sys/devices/platform/tm1668/text. При их отсутствии демон должен завершить работу.

Демон создает и слушает Unix сокет на прием команд (/var/run/frontpanel).
При выводе текста/слова, в котором более 4 знаком, должна осуществляться прокрутка сообщения. Интервал движения текста должен настраиваться.
Сперва показываются первые 4 символа текста на время t1 милисекунд. Далее идет сдвиг на одну позицию. Отображение сдвинутого текста происходит на другую величину: t2 милисекунд. Сдвигаемый текст отображается до тех пор, пока не отобразится пустота (причем пустота должна вывестись длительность t2). Далее всё заново, если не исчерпался запас времени. Таким образом можно подсчитать, что строчка длиной n (n>4), будет отображаться по времени t1+t2(n-1) милисекунд.
Вывод часов должен происходить так: обновление каждую секунду + моргание двоеточия (одну секунду не отображается, следующую отображается)

Команды имеют однобуквенные и длинные аналоги.
Описание команд, которые принимает и понимает демон из сокета:
s/status <CODE_NUM>[:<TIME>] <message>
		- вывести статусное сообщение (сообщение которое гарантированно будет отображаться указанное количество секунд). Ничего не возвращает.
		<CODE_NUM> - код по которому можно снять статусное сообщение, указывается отправителем команды,
		<TIME> - период времени в секундах, в течении которого будет отображаться сообщение,
		<message> - отображаемое сообщение.
n/notify <TIME> <message>
		- показать уведомление (может заменятся любым другим уведомлением или статусом). Ничего не возвращает.
		<TIME> - период времени в секундах, в течении которого будет отображаться сообщение,
		<message> - отображаемое сообщение.
?/busy
		- возвращает количество оставшегося времени до освобождения индикатора от статусных сообщений, в милисекундах. 0 - индиктор свободен.
t/time <enable>
		- команда включающая/выключающая индикацию времени (по умолчанию включена). Ничего не возвращает.
		<enable> - 0 - выключить, 1 - включить.
d/delays <t1> <t2>
		- Задание параметров t1 и t2.
b/bright <brightness>
		- Управление яркостью от 0 (выключено) до 8.
p/pulse <enable>
		- Включение/выключение режима пульсации.

Очистка индикатора осуществляется с помощью команды: s<CODE_NUM>:0 "".
<message>, помимо обычного текста, может передаваться в специальном формате:
         "xx:xx" - выведет помимо знаков еще и двоеточие. Размер сообщения 3-5 символов, 3-ий символ обязательно равен ':'.
